## About the Project
Bankist is a vanilla JavaScript web based fictional bank application with a minimalist interface developed using arrays and advanced DOM manipulation techniques. Multiple real-life banking actions such as user authentication, loan requests, money transfers between accounts and account closure are simulated within the application.  

The application consists of two pages:

1. The home page: Website landing page
2. App Page: Visible upon logging in and contains functionalities such as money transfers between accounts, loan requests and account closure. Users can also view balances and review recent transactions.


## Built with

- HTML
- CSS
- JavaScript

## Usage
Follow the following steps in order to run Bankist home and app page locally:

1. Clone the repository:
`https://github.com/JucalisLife/Bankist.git`
2. Navigate to the project directory:
`cd Bankist\home` or `cd Bankist\app`
3. Open index.html in your preferred web browser.
